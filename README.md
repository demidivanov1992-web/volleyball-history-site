# ValleyballHistory.com — статический сайт

Этот проект уже подготовлен как статический сайт для GitHub Pages.

Используемые файлы:
- `index.html`
- `history.html`
- `training.html`
- `players.html`
- `styles.css`
- `images/`
- `videos/`

## Как публиковать

1. Залей репозиторий на GitHub.
2. В настройках GitHub Pages выбери ветку (`main` или `gh-pages`) и корневую папку (`/root`).
3. Если хочешь собственный домен, добавь `CNAME` через настройки GitHub Pages.

## Что важно

- Всё подключается напрямую: `images/...`, `videos/...`, `styles.css`.
- Flask не нужен и не используется для финальной версии сайта.
- В корне расположены готовые HTML-страницы.

## Локальный просмотр

Для локальной проверки просто открой `index.html` в браузере.
